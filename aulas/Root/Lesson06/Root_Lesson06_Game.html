Real Estate Pitch Battle

In each round, one student becomes the real estate agent and the others become buyers. The agent must sell a house or apartment by describing the location, rooms, furniture, and object positions using there is / there are and prepositions of place.

This game is designed for online classes with one shared screen. Only the teacher needs to click buttons, control the timer, award points, and confirm whether the property was sold.

  • Seller mission: explain where the property is, how many rooms it has, and what furniture is inside.
  • Buyer mission: ask questions like “Is there a balcony?” or “Where is the sofa?”
  • Scoring focus: longer, clearer descriptions earn more points. Sales give an extra bonus.

Add 2 to 5 players, choose the number of rounds, and start the online game.
